Android on BUG 2.0

Run Android on your BUG 2.0 device.

This is the start page for the Bug Labs-based port of Google's Android to the BUG 2.0 device.

This project can be used to hack on the Android platform internals as well as run Android software on BUG hardware. This is an open project that welcomes contributions, and utilizes the transparent GitHub project infrastructure for source control and issue tracking.

Install Android Binaries

For those wishing to simply run Android on the BUG device, go to the build drops page hosted on bugcommunity.com: http://bugcommunity.com/downloads/files/android_builds

Create Bootable Android SD Card

Here is how you can write it to an SD card such that it can be booted: 
$ git clone xxx
$ cd android/scripts
$ wget [URL to IMAGE]
$ ./create-sdcard.sh [CARD READER DEVICE] [IMAGE]

Now, after the writing is complete, put the card in the BUG and boot it.

Build Android Source

By using the repo command, you can checkout all the projects necessary to build Android on BUG with:

$ repo init -u git@github.com:buglabs/android.git

For detailed instructions on building from source, check this page: http://wiki.buglabs.net/index.php/Experimental:Building_Android_image_for_BUG_YT.
